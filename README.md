# BeQuiet

<a href="https://github.com/Alterux/BeQuiet/raw/master/BeQuiet-1.0.zip">Download latest version </a>

Minecraft resource pack that makes certain sounds less loud.

Sounds included in v1.0:
- Wither spawn & death
- Ender Dragon death & growls
- Endermen death

- Minecart sounds

- Thunder sounds
